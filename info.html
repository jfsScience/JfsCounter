PulseCounterESP

A programm for a special hardware and purpose

About

A pulse counter based on a ESP32 with a OLED display. It counts the nummer of pulses during a fixed time frame The idea was to build a small and cheap counter as an interface between a device witch genenerates 5V TTL an a GUI that runs on a PC.

Usage

After installation the software on the PC and connecting the ESP32 via USB the computer you can run a first test:

  • Select the COM port and open it.
  • Connect input and output of the device with a BNC cable.
  • Select the Timebase or 100ms as default.
  • Hit start.
  • Check Testlauf [on] and the oscillater will increment the output frequency.
  • to adjust the display hit [clear Display].
  • After a while hit [stop]. With the right mousebutton over the display you can select a lot of options that are included.
